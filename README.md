## 🎯 What CHAARI_2.0 does

CHAARI_2.0 is a voice-based AI app that helps you control your computer with speech.

It is built for privacy first. It runs local AI parts on your device, uses signed commands, and checks each action before it runs.

It speaks Hinglish natively, so you can talk in a mix of Hindi and English.

Use it to:
- open apps
- search and fetch info
- run simple desktop actions
- talk to a local AI helper
- keep control with safety checks in place

## 🛡️ Safety and privacy

CHAARI_2.0 uses a safety pipeline before it acts on your computer.

That means it checks commands before execution and blocks unsafe requests.

It also uses:
- signed commands with RSA-2048
- replay-protected nonces
- a two-node cryptographic mesh
- local-first AI flow where possible

This helps keep control on your side.

## ❓ Common problems

### App will not open
- Right-click the app and choose Run as administrator
- Check that Windows did not block the file
- Re-download the file if it looks incomplete

### Mic does not work
- Open Windows Settings
- Go to System > Sound
- Pick the right input device
- Test the mic with another app

### Voice input is not heard
- Turn up mic gain in Windows
- Move closer to the microphone
- Close other apps that use the mic

### App feels slow
- Close other heavy apps
- Restart Windows
- Make sure you have enough free RAM
- Use a lighter local model if the app offers that option
